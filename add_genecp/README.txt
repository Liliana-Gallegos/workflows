To add gen or genecp text on input COM gaussian files see .txt files for examples.

1. A text file is required with the following format:
	genecp==(Elements for the gen/genecp text below)

	#genecp text below
	(basis set)
	(gen/genecp text here)

2. Run command: python add_genecp.py *.com --txt (genecp_filename).txt
